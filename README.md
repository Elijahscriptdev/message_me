# Project: message_me app
This is a web app for chatting(sending messages)

## Built With

- Ruby v2.7.0
- Ruby on Rails v5.2.1

## Technologies
- HTML
- CSS
- Ruby
- Ruby On Rails
- SASS
- Semantic UI

## Prerequisites

- Ruby: 2.7.0
- Rails: 5.2.1

## Functionalities 
- Signup
- Login
- Send a chat message

## Setup

```sh
$ git clone https://github.com/Elijahscriptdev/message_me.git
```

Navigate to the extracted folder
```sh
$ cd message_me
```

Install the gems
```sh
$ bundle install
```

Run the migrations
```sh
$ rails db:migrate
```
### Usage

Start server with:

```sh
    rails server
```

Open `http://localhost:3000/` in your browser.
